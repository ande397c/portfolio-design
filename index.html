Design for Developers - design and UI

About this theme

This theme has been focusing on improving the design and UI. Among other by diving into the use of fonts, colors, animations, and responsive design. These subjects have furthermore been used in different assignments. Now it´s time for you to explore it all.

Assignments for this theme

Fonts and UI cards

This assignment was to create a mockup landing page design using fonts and gray scales only.

Colors and animations

The assignment is to apply a color pallet to a mockup landing page design, then add animations.

Responsive design

In this assignment, the goal is to transform a mockup landing page into a responsive layout.

Designing with frameworks

This assignment was to do visual design using a framework only.

My thoughts on the theme

This theme has served as a nice addition to 5 weeks of just pure javascript. In context to these weeks, working with design and UI has been refreshing. Besides that, a basic understanding of these areas are important to a frontend developer.
