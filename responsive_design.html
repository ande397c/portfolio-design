Making a responsive layout

A responsive layout is a very big part of modern web-development. This assignment focused on the use of CSS Flex, Grid, min width/height and media queries to layout the page.

Starting from scratch

The overall goal of the assignment was very clear. However, by starting with a full html document and an empty CSS proved more challenging than first imagined. Simply because I didn't have a visual representation of what the layout should look like. Instead, we were provided with comments in the HTML document to help guide us.

The final product

The final product fills the role of a responsive web layout. I took advantage of the HTML and CSS that we had used throughout the course. The overall page generally follows a classic two column grid. The assignment build on what we had learned earlier and spanned upon the general use of CSS grid and flex, that we had used before.
