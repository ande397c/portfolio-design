Introduction

The assignment was to create a mockup landing page design using fonts and grey scales only. Fonts and UI cards consist of two parts; first part is to implement a font hierarchy and the second part is to design UI cards.

Creating a font hierarchy

To complete this part I used type-scale.com. This helped me test the hierarchy beforehand. I used 'Forum' as my display font and 'Lato' as my paragraph font. The display font is only used as a h1 heading. For smaller headings I used Lato and adjusted the font weight to match the layout.

Improving the UI Cards

Working with this part required following the guidelines for designing UI cards and using CSS techniques that we have learned in class. For instance, I used contrast and proximity to draw the user's eye to the most important thing first.
